# Atividades de Programação Orientada a Objetos (POO) em Python

Este repositório contém as atividades desenvolvidas para o aprendizado e aplicação dos conceitos de Programação Orientada a Objetos (POO) utilizando a linguagem Python. As atividades cobrem desde a criação de classes até conceitos mais avançados como herança, polimorfismo, encapsulamento e abstração.

## Descrição das Atividades

As atividades abordam os seguintes tópicos:

1. **Criação de Classes e Objetos**: Introdução ao conceito de classe e objeto, incluindo como instanciar objetos e definir atributos e métodos.
2. **Encapsulamento**: Implementação do controle de acesso a atributos e métodos, utilizando propriedades e getters/setters.
3. **Herança**: Criação de hierarquias de classes para reutilização de código e modelagem de relacionamentos.
4. **Polimorfismo**: Uso de polimorfismo, tanto no contexto de sobrecarga de métodos quanto substituição de métodos nas subclasses.
5. **Abstração**: Desenvolvimento de classes abstratas e interfaces para garantir que as subclasses sigam um contrato.
6. **Composição e Agregação**: Como representar relacionamentos entre objetos por meio de composição e agregação.

Cada atividade é apresentada em arquivos separados, com explicações e exemplos para serem seguidos.
